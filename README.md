# computer-graphics-project
Codelyoko
## Tasteri

* `WASD` - kretanje
* `ESC` - zatvaranje prozora
* `B` - Blinn-Phong model osvetljenja
## Tehnike koriscenje u projektu

* Shaders
* Textures
* Transformations
* Camera
* Lighting
* Model loading
* Blending
* Face culling
* Advanced lighting
* Cubemaps
